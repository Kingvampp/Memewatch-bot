# MemeWatch Bot 🚀

A Discord bot that tracks memecoin prices and provides detailed token information.

## Features

- 💰 Real-time token prices and market data
- 📊 Market cap and volume information
- 📈 Price change tracking with 24h changes
- 🏆 All-Time High tracking with dates
- 🔗 Smart contract addresses for different chains
- 🌐 Social media links (Twitter/X, Telegram)
- 📱 Trading platform links (DEXScreener, Birdeye, BullX)

## Usage

Simply use `$` followed by:
- Token name: `$pepe`, `$bonk`
- Contract address: `$0x...`

## Supported Chains

- Ethereum (ETH)
- Solana (SOL)
- Binance Smart Chain (BSC)

## Data Sources

- CoinGecko API
- DEXScreener
- Birdeye (for Solana)
- BullX
